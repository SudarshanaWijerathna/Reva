# Commit Message Guidelines

Format:
type: short description

Allowed types:
- feat
- fix
- docs
- refactor
- model
- data
- chore

Example:
feat: add chatbot intent classification


| Type       | Meaning          |
| ---------- | ---------------- |
| `feat`     | New feature      |
| `fix`      | Bug fix          |
| `docs`     | Documentation    |
| `refactor` | Code improvement |
| `model`    | ML model changes |
| `data`     | Dataset changes  |
| `chore`    | Setup / config   |
